PHILOSOPHY

「想定外」は、
いちばん正直なフィードバック。

展示において、ユーザーが私の制作したコントローラーを
想定外の持ち方や遊び方で扱う場面に何度も直面した。
しかし客観的に観察すると、それらの行動は極めて自然で理にかなっていた。
この経験から、作り手の「こう遊ぶべき」という無意識のバイアスを自覚し、
初見プレイヤーの直感的な行動を観察・予測して、
自然な形で面白さへ導くアフォーダンスとルール設計の重要性を学んだ。

実証としての作品

思想は作品でしか証明できない。デバイスの実機、プレイの現場、企画書。3つの視点から各プロジェクトを記録しています。

PROJECT AISCA2026 提出目標

汲・撒・響

柄杓型・桶型の独自デバイスで、水を「汲む・撒く」物理的な解放感をそのままゲームに写し取る作品。床へモニターを水平配置することで、2人のプレイヤーが画面を挟んで自然と向かい合い、対面コミュニケーションが誘発される空間設計を狙っている。

PROJECT BBitSummit2026 出展

『ニギ/ニギっとスライム研究所』専用コントローラー「ニギコン2」

「握り込む」という一動作の手応えを、ゲーム内アクションへ直接連動させた専用コントローラー。握る強さ・速さがそのままスライムの変形に反映され、入力と結果のあいだに翻訳を挟まないダイレクトな体験を設計した。

PROJECT C遊創研/GCCP・BitSummit2027 出展目標

ShakeCon

内部に独立稼働するパーツを組み込み、振るとカチャカチャと物理的な音が鳴るコントローラー。「振る」行為そのもののプリミティブな面白さを起点に、音と手応えがゲーム体験へ還流する設計を探求している。

マイコン内部ギミックimages/shakecon-inside.jpg
シリアル通信連携仕様書images/shakecon-spec.jpg

* このカードにマウスを乗せると実機のように傾き、カチャカチャ震えます

PROJECT D授業内制作（アナログ）

トリ合い譲り合い

焼き鳥の最後の1本をテーマにした、譲り合いの駆け引きを楽しむ社会人向けカードゲーム。「奪う」ではなく「譲る」をプレイの主語に置くことで、場の空気の読み合いがそのままゲームメカニクスになる構造を設計した。デジタルに限らない遊びの幅を示す制作実績。
